League News

Description

The basis of the project is to give a personalized news source for fans of the NFL. It requires an account in which first name, last name, username, password and a selection of a favorite team are required to access the website. It is possible to login using username and password. Using the favorite team selection, the tweet list display and the styling of the page will update accordingly giving a proper news outlet. Also, it allows you to submit an article to be posted if you would like.
